# Chemistry Learning Hub

A comprehensive, beautifully designed interactive chemistry learning hub for students and children.

## What it includes

- **Learning Studio** with core Gen Chem + Chem II topic explainers.
- **Mini Periodic Explorer** for high-frequency element review.
- **Calculation Center** with fast calculators for repetitive and tedious chemistry work:
  - Molarity
  - Dilution
  - Ideal Gas Law (solve for moles)
  - pH from hydrogen ion concentration
  - Stoichiometric mass conversion
  - Heat transfer (`q = mcΔT`)

## Run locally

Because this is a static web app, you can open `index.html` directly, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Vision

This project is designed to evolve into an all-in-one chemistry mastery platform and go-to calculator resource for Gen Chemistry and Chem II students.
